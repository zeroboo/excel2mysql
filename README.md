## Installation-instructions
From npm: `npm install xlsx2mysql`
## Usage
Require module:<br>
`const excel2Mysql = require("xlsx2mysql");`

To generating mysql import queries:<br>

`excel2Mysql.exportExcelToQuery(excelFile, config, sqlFile);`
